## 🚀 Current Research

### Privacy-Preserving Fall Detection with 3D LiDAR

My current work investigates **human fall detection and activity monitoring using real-world 3D LiDAR point clouds**.

Unlike conventional RGB-camera monitoring, LiDAR captures geometric information about the environment without recording traditional images, making it attractive for privacy-sensitive applications.

A simplified pipeline:

```text
          3D LiDAR Sensor
                 │
                 ▼
          Raw Point Cloud
                 │
                 ▼
       Human / ROI Extraction
                 │
                 ▼
      Point Cloud Preprocessing
                 │
                 ▼
       Deep Learning Encoder
                 │
                 ▼
      Fall / Non-Fall Decision
                 │
                 ▼
     Evaluation + Visualization
```

I am also exploring **self-supervised representation learning** to improve learning from unlabeled real-world LiDAR data.
